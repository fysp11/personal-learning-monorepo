# Audio Overview 2: AI Design Patterns

## Purpose

A 15-minute NotebookLM audio overview covering **AI design patterns** — confidence routing, staged workflows, and multi-market architecture. Best consumed while reviewing the live-round scenarios.

## What It Covers

1. The 5-stage accounting pipeline: extract → categorize → VAT → route → book
2. Why each stage must fail and be tested independently
3. Confidence routing in depth — ECE under 0.05, per-stage thresholds
4. The 5-level automation maturity ladder applied to Germany → France
5. Market config as data, not code — why VAT rates must be code
6. Earned autonomy vs claimed autonomy
7. AI team vs ML team distinction (Q8)
