Create a professional infographic that works like a one-page risk-controls report for the Finom interview prep.

Goal:
- summarize the control model the candidate should articulate for AI accounting and other correctness-sensitive workflows

Audience:
- the candidate, using this as a visual memory aid before the interview

Content to include:
- workflow stages:
  - intake
  - extraction
  - classification
  - reconciliation
  - proposal
  - approval / exception routing
  - finalization / audit
- key control mechanisms:
  - grounding in evidence
  - workflow constraints
  - confidence policy
  - human review
  - observability
  - rollback / containment
- top failure modes:
  - false confidence
  - silent mismatch with transaction context
  - edge-case brittleness
  - weak observability
- staged rollout:
  - shadow mode
  - draft mode
  - approval-gated action
  - selective automation

Style:
- clean, professional, report-like
- avoid playful or decorative visuals
- optimize for fast scanning

Layout:
- portrait
- standard detail
- use sections, arrows, and compact labels

Make sure the infographic emphasizes:
- the product is the controlled workflow, not the raw model output
- one global confidence score is not enough
- trust is earned through routing, evidence, review, and auditability
