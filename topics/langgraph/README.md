## Scope

- graph-based agent workflows
- persistence, memory, and interrupts
- human-in-the-loop patterns
- MCP and tool integration
- starter repos and implementation references

## Layout

- `sources/firecrawl/` - harvested docs and repo snapshots grouped here so they are searchable under a first-class topic

## Current Source Pack

## Why This Exists

The repo already had a sizable LangGraph source cluster, but it lived under `.firecrawl/` without a topic home. This folder makes the subject explicit in `topics/` and easier to resume later.
