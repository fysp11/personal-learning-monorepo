# Codex

This topic is the workspace for learning how to use Codex effectively for study, implementation, debugging, and multi-agent execution.

## Scope

- Codex operating patterns
- code-agent delegation and review posture
- workflow design for study, test, and code-heavy sessions
- comparisons with adjacent coding agents

## Layout

- `notes/` - durable notes about Codex and closely related coding-agent usage patterns

## Why This Exists

Codex already appeared in the learning collection as first-class knowledge, but part of that material was still sitting outside `topics/`. This topic makes it explicit and resumable.
