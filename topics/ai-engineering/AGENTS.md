# AI Engineering Agent Instructions

- Keep this folder focused on learning outcomes, not ad-hoc notes. Use `prep/` for sprint-specific memo, `resources/` for references, `experiments/` for proof-of-concepts, and `artifacts/` for reusable outputs.
- Always summarize any new learning or experiment in a markdown note under the appropriate folder; include date and short conclusions.
- If you create NotebookLM prompts/outputs, place them under `artifacts/notebooklm` (create the subfolder when needed) and reference the prompt in `README.md`.
- Avoid mixing company-specific prep with this topic. Keep the topic self-contained so it can be resurrected independently later.
