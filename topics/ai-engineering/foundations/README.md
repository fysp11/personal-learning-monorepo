# AI Engineering Foundations

This folder holds durable foundational notes that were previously stored as shared extracted entities.

## Structure

- `architectures/` - system shapes and orchestration boundaries
- `concepts/` - core ideas and definitions
- `methodologies/` - measurement and evaluation methods
- `patterns/` - reusable design patterns
- `products/` - product concepts used as reference anchors
- `systems/` - system-level notes that fit AI engineering more than a separate tool topic

## Why This Exists

These notes are more useful when they live inside the AI engineering topic tree instead of sitting outside `topics/`.
